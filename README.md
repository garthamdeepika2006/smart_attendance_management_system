# Smart Attendance Management System

A Flask and MySQL based Smart Attendance Management System that enables student registration, attendance marking, attendance tracking, and dashboard analytics.

## Features

- Student Registration
- Face Capture
- Attendance Marking
- Teacher Dashboard
- Present/Absent Reports
- Token Based Attendance
- Student Management
- Attendance Analytics

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository

```bash
git clone https://github.com/garthamdeepika2006/smart_attendance_management_system.git
```

2. Navigate to project folder

```bash
cd smart_attendance_management_system
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Configure MySQL database

5. Run the application

```bash
cd backend
python app.py
```

6. Open browser

```text
http://127.0.0.1:5001
```
